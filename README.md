# NokiLZT / NokiaChat

![Preview](https://cdn.rszmeow.pro/b819571927f3.jpg)

Пользовательский скрипт для Tampermonkey (lolz.live / lzt.market).

## Установка

1. Установите расширение **Tampermonkey** в браузер.
2. Откройте Tampermonkey и нажмите **Create a new script**.
3. Удалите шаблонный код и вставьте содержимое файла `NokiaChat.js`.
4. Сохраните скрипт (`Ctrl+S`).
5. Откройте сайт `https://lolz.live/` или `https://lzt.market/` и обновите страницу.

## Обновление

1. Откройте скрипт в Tampermonkey.
2. Замените код на новую версию из `NokiaChat.js`.
3. Сохраните и обновите страницу сайта.

## Примечание

Скрипт работает только на доменах, указанных в заголовке `@match`.
